# 몬스터 디펜스 (Monster Defense) 게임 추가 계획

작성일: 2026-03-22

---

## 게임 컨셉

**장르**: 360도 서바이벌 디펜스
**타겟**: 초등학생+
**조작**: 클릭/터치 (자동 공격 무기 + 수동 업그레이드)

### 핵심 규칙
- 화면 중앙에 기지(성 🏰), 사방에서 몬스터 접근
- 무기가 자동으로 가장 가까운 몬스터 공격
- 몬스터 처치 → 골드 💰 획득
- 골드로 무기 업그레이드 (속도/파워/사거리) 또는 신규 무기 구매
- 기지 HP 0 = 게임 오버
- 웨이브 시스템 (5웨이브마다 보스)

### 무기 4종
- 🏹 화살탑: 기본, 빠른 단일 공격
- 💣 캐논: 느리지만 범위(스플래시) 공격
- ⚡ 마법탑: 관통 공격 (여러 적 관통)
- ❄️ 얼음탑: 슬로우 효과

### 몬스터 5종
- 🟢 슬라임: 기본, 느림
- 🦇 박쥐: 빠름, 체력 낮음
- 🪨 골렘: 느림, 체력 높음
- 👻 유령: 중간, 가끔 투명(공격 회피)
- 🐉 드래곤: 보스 (5웨이브마다)

### 업그레이드 시스템
- ⚡ 공격 속도 (50G → 100G → 200G...)
- 💥 공격력 (50G → 100G → 200G...)
- 📏 사거리 (80G → 160G...)
- 🆕 무기 추가 (300G → 600G → 1200G...)

### 웨이브 설계
- 웨이브 1~5: 슬라임만 (입문)
- 웨이브 6~10: 박쥐 추가
- 웨이브 11~15: 골렘 추가
- 웨이브 16+: 유령 추가, 매 웨이브 체력/속도 +5%
- 매 5웨이브: 보스 드래곤 🐉
- 웨이브 사이 5초 휴식 (업그레이드 시간)

### 이펙트
- 무기 발사 시 탄환 애니메이션 (화살/포탄/번개/얼음)
- 몬스터 피격 시 데미지 숫자 팝업
- 처치 시 파티클 이펙트
- 보스 등장 시 화면 흔들림 + "BOSS!" 경고
- 웨이브 클리어 시 팡파레
- 골드 획득 시 +숫자 플로팅

## 기술 노트

- CSS div 기반 렌더링 (기지 중심 상대 좌표)
- GAME_W = 500, GAME_H = 500 (정사각형 필드)
- useGameScale, useTouchLock 훅 활용
- deltaTime 기반 게임 루프 (프레임 독립적)
- 탄환/몬스터는 배열 상태, 매 프레임 위치 업데이트
- 몬스터 최대 50마리 동시 (성능 제한)
- 무기 슬롯: 기지 주변 8방향 (N, NE, E, SE, S, SW, W, NW)
- 모바일: 터치로 업그레이드 패널 조작
